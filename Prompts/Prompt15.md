## Prompt 15 — Guía de instalación

```
Redacta la "Guía de Instalación" del proyecto como un documento independiente, que
incluya, en este orden:

1. Requisitos previos: Visual Studio (versión recomendada), .NET SDK instalado, SQL
   Server, SQL Server Management Studio (SSMS).
2. Cómo restaurar/abrir el proyecto en Visual Studio.
3. Cómo ejecutar el script RentaDepartamentos.sql en SSMS para crear la base de datos y
   las tablas (creado en el Prompt 2).
4. Cómo configurar la cadena de conexión en appsettings.json para que coincida con el
   entorno local de cada integrante del equipo.
5. Cómo compilar y ejecutar el proyecto (botón de inicio en Visual Studio o comando
   "dotnet run").
6. La URL local donde se visualiza la aplicación una vez en ejecución (por ejemplo,
   https://localhost:xxxx).
7. Solución a problemas comunes de conexión a SQL Server (por ejemplo, error de
   Trusted_Connection, instancia de SQL Server no encontrada, puerto bloqueado).

Usa un lenguaje técnico pero claro, en formato de pasos numerados, pensado para que otro
desarrollador (no necesariamente el equipo original) pueda instalar el proyecto desde
cero sin ayuda adicional.
