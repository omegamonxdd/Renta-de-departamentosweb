## Prompt 16 — Verificación final contra la rúbrica de evaluación

```
Actúa como revisor de calidad del proyecto RentaDepartamentosWeb. Verifica el proyecto
completo contra esta rúbrica de evaluación y, para cada criterio, indica si se cumple,

sugerencia concreta de qué agregar o corregir:

| Criterio                          | Valor |
|------------------------------------|-------|
| Página web funcional               | 15%   |
| Conexión a SQL Server              | 10%   |
| Consultas SQL desde C#             | 15%   |
| CRUD de departamentos              | 20%   |
| Uso de clases, métodos y POO       | 10%   |
| Aplicación básica de SOLID         | 10%   |
| Buenas prácticas y Clean Code      | 10%   |
| Manual de usuario                  | 5%    |
| Guía de instalación                | 3%    |
| Trabajo colaborativo               | 2%    |

evidencia:
- Proyecto web en .NET funcionando.
- Script .sql de la base de datos.
- Manual de usuario.
- Guía de instalación.
- Código documentado (comentarios donde son necesarios, nombres significativos).
- Espacio para capturas de pantalla.
- Espacio para evidencia de trabajo colaborativo del equipo (por ejemplo, historial de

  repartidas).
```
