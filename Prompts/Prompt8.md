## Prompt 8 — Vistas Razor: Página principal

```
Crea la vista principal del sitio (Views/Home/Index.cshtml o la página de inicio que
prefieras dentro de la estructura del proyecto). Debe:

- Mostrar una breve descripción del sistema ("Sistema Web de Renta de Departamentos").
- Tener accesos directos (botones o tarjetas) a:
  1. Departamentos disponibles
  2. Registro de departamento
  3. Administración de departamentos (lista general)
  4. Búsqueda de departamentos

Usa Bootstrap (ya incluido por defecto en las plantillas de ASP.NET Core MVC) para un
diseño limpio y responsivo, con un estilo simple tipo "panel administrativo". No uses
frameworks adicionales ni dependencias externas más allá de lo que trae la plantilla
estándar de ASP.NET Core MVC.
```
